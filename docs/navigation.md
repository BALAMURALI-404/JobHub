# Job Hub - Navigation Structure

## 1. Authentication
- **Login** (`/login`)
  - **Purpose**: Allows users to log in to their existing account.

- **Signup** (`/signup`)
  - **Purpose**: Allows users to register for the platform (with a unified account system).

---

- **Search** (`/search`)
